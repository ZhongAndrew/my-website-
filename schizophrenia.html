常見的心理疾病

思覺失調症

什麼是思覺失調症？

思覺失調症是一種嚴重的精神疾病，通常在青春期或成年早期發病，影響約1%的人口。
該疾病的病程長期且慢性，常見於所有社經地位的群體，但社經地位低者更易發病。
思覺失調症有高度的家族遺傳傾向，病程一般分為療效佳、部分緩解和需要長期療養三種情況。

心理健康指南

1. 維持規律的生活作息：建立穩定的生活和工作環境，有助於患者穩定情緒。

2. 遵守治療計劃：按時服用藥物，並定期就醫，保持與醫療專業人士的良好溝通。

3. 尋求支持：與家人、朋友保持聯絡，參加支持小組，獲得情感支持和資訊。

4. 學習應對技巧：學習應對壓力和困難的技巧，改善生活質量。

疾病症狀

1. 思考障礙 :

答非所問、語無倫次。

妄想：被害妄想、關係妄想、被控制妄想等。

2. 知覺障礙 :

聽幻覺、視幻覺或體幻覺，可能有自言自語或手舞足蹈的行為。

3. 情感障礙 :

表情平淡、不適當，情緒變化劇烈。

4. 行為異常 :

怪異和混亂的行為。

5. 負性症狀 :

對事物失去興趣、缺乏動機、社交退縮、思考貧乏。

解決疾病方法

1. 藥物治療 :

主要使用抗精神病藥物。新一代藥物副作用較少，能改善生活品質。

遵循醫師指示服藥，若出現副作用應與醫師討論調整。

2. 心理社會治療 :

包括個別或家族治療，重點在於教育、指導、支持、認知行為與現實取向。

提供康復之家、職能復建、庇護工廠/商店、居家治療等支持服務。

3.早期介入與管理 :

在急性期專注於急性症狀的治療，穩定期則降低壓力，鼓勵社會參與和維持角色功能。

4. 家屬支援 :

降低對病人的期待，準備應對疾病復發的心理準備。

發展照顧病人的技巧，維持自己的生活空間，避免孤立，與其他家人溝通，共同承擔。
